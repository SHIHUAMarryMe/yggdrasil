1 把xls 另存为txt(用制表符分隔) 
	动画和宠物播放位置的xls文件(example: sp_tab.xls) 另存为 sp_tab.txt(必须是这个名字)
	特效描述的xls文件(example: skill_anim_tab.xls) 另存为 skill_anim_tab.txt(必须是这个名字)
	特效描述的xls文件(example: skill_tab.xls) 另存为 skill_tab.txt(必须是这个名字)

2 在该两个文件的所在目录下 运行 tool_of_skill_table.exe 生成的 skill.xml skill_anim.xml 就是结果的文件


技术部注意： 该程序编译需要tinyxml 该lib在yggr/tools里
